<lore>
A blank board is just potential. Etch a purpose into it under the laser and it becomes the mind of a working robot.
</lore>
<no_lore>
The Programming Table turns a blank redstone board into a programmed board that defines a robot's behavior. It is laser-powered: choose a board type and the table crafts it once enough MJ has accumulated.
</no_lore>
<chapter name="Information"/>
Insert a blank redstone board into the <bold>input</bold> slot. The <bold>board options</bold> list every behavior the table can program — click one to select it. The <bold>progress</bold> bar shows how much MJ has been stored toward that board; when it fills, the finished programmed board drops into the <bold>output</bold> slot. Pull it out with a pipe or by hand.
A programmed board is combined with a blank robot at an <link inline="buildcraft:block/integration_table"/> to build a working robot — picking up items, farming, mining, delivering, and so on. See <link inline="buildcraft:concept/robots_and_zones"/> for how robots and their boards fit together.

<chapter name="Power"/>
Like the <link inline="buildcraft:block/assembly_table"/>, the Programming Table is a <link inline="buildcraft:block/laser"/> target. Aim one or more lasers at it to fill the progress bar; more lasers program boards faster.
