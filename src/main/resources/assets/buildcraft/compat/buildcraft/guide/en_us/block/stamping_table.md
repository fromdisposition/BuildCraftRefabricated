<no_lore>
The Stamping Table is a laser-powered single-item processor. It consumes MJ from adjacent lasers to stamp a registered input item into its output — a simple one-in, one-out companion to the <link inline="buildcraft:block/assembly_table"/>.
</no_lore>
<chapter name="Information"/>
Insert the item to be stamped into the <bold>input</bold> slot. The table consumes MJ from an adjacent laser to process it, and the result lands in the <bold>output</bold> slots. Pull the output out with a pipe or by hand to keep the table running.
Which items can be stamped, and what they produce, is defined by recipes — JEI shows the input, output, and MJ cost for every stamping recipe.
<recipes_usages stack="buildcraftsilicon:stamping_table"/>
<chapter name="Power"/>
Like the <link inline="buildcraft:block/assembly_table"/>, the Stamping Table is a <link inline="buildcraft:block/laser"/> target. Aim one or more lasers at it; more lasers mean faster stamping.
