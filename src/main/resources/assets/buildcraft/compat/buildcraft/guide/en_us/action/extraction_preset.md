<lore>
The Emzuli Transport Pipe keeps several extraction filters ready; a gate picks which one is active.
</lore>
<no_lore>
Extraction Preset is a gate action family on Emzuli Transport Pipes. Each preset corresponds to one of the pipe's coloured filter slots.
</no_lore>
<chapter name="Behaviour"/>
When activated, the gate tells the Emzuli pipe to use the selected preset colour for extraction from inventories. Without a preset selected by the gate, the Emzuli pipe cannot extract items.
