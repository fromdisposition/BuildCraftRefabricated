<lore>
Fold a handful of loose items into a neat paper parcel, ship it down a pipe, and unfold it wherever you need it.
</lore>
<no_lore>
The Packager folds items into a single package item using paper, following a configurable 3×3 pattern. Packages travel through pipes as one stack and can be unpacked later, back into their original items.
</no_lore>
<chapter name="Information"/>
The GUI has three parts. The <bold>pattern</bold> is a 3×3 grid: click a cell to toggle it, or click while holding an item to set a ghost template. Lit cells mark which storage slots get packed. The pattern maps directly onto the nine <bold>storage slots</bold> below it — the item in each enabled slot is folded into the package.
Insert <bold>paper</bold> into the input to fold the configured pattern into a package; the finished package appears in the output. Insert an existing package into the input instead and it is unpacked, returning its contents to the storage row.
Items in slots enabled by the pattern are consumed when a package is created. Feed the input and pull the output with pipes to run it hands-free.
